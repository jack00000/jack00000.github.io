4.常用文件输入输出类

  • BufferedReader

  • System.getProperty(“java.io.tmpdir”) java运行产生的临时文件目录

2.调试视图区域介绍

    1. 当前是哪个线程，因为是非多线程程序，所以就是主线程
    1. 对第八行运行有影响的几个变量的值，这个就是调试的主要作用，观察这些值的多少，进行分析问题所在或者理解代码逻辑

3.注意点

  • 2.左边的线程太多 直接邮件terminal all 再debug 一次就清除啦
